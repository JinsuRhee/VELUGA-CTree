Output data format
==================

This page describes the output products generated by **VELUGA-CTree**.

1) Data Format
--------------

Overview
~~~~~~~~

The outputs are written to the directory specified by ``output_dir``.
Files may be produced per snapshot and/or per processing stage, depending
on your configuration.

.. note::

   Please update the filenames, groups, and dataset names below to match
   the current implementation.

File layout
~~~~~~~~~~~

.. code-block:: text

   <output_dir>/
     ├─ logs/
     │   └─ veluga_ctree.log
     ├─ branches/
     │   ├─ branch_<snap>.h5
     │   └─ ...
     ├─ complete_tree/
     │   ├─ ctree.h5
     │   └─ ctree_summary.txt
     └─ ...

Main file(s)
~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 35 20 60

   * - Item
     - Type
     - Description
   * - ``ctree.h5``
     - HDF5
     - Main complete-tree output (recommended for analysis).
   * - ``branch_<snap>.h5``
     - HDF5
     - Intermediate branch output written per snapshot (optional).
   * - ``ctree_summary.txt``
     - text
     - Human-readable summary/statistics (opt
